# LibraryManager
 This project is an apirest to manager a library,
 targeting only for back-end.
 
# IDE
I recommend use Intellij Community version  
`` https://www.jetbrains.com/pt-br/idea/download/?section=windows ``

# How Install Dependencies
Using Intellij is only need to open the project and the IDE will read the ``pom.xml`` and automatically install all dependencies.
